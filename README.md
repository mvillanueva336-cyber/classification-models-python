# Customer Churn Classification Models

This project implements and compares multiple machine learning classification algorithms for predicting customer churn using Python.

## Project Overview

Customer churn data was preprocessed and divided into training and testing sets. Several classification models were trained and evaluated to determine which algorithm best predicts customer attrition.

Performance was measured using confusion matrices, classification reports, ROC curves, and Area Under the Curve (AUC).

## Models Implemented

* Logistic Regression
* Linear Discriminant Analysis (LDA)
* Lasso Logistic Regression
* K Nearest Neighbors (KNN)

## Hyperparameter Tuning

* Grid Search Cross Validation
* 10 Fold Cross Validation

## Evaluation Metrics

* Accuracy
* Confusion Matrix
* Precision
* Recall
* F1 Score
* ROC Curve
* Area Under the Curve (AUC)

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

Customer churn dataset from the `modeldata` package.
